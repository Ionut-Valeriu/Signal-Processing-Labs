This repository contains lab work, assignments, and project material for the Signal Processing (Procesarea Semnalelor) course in the 3rd year of the Computer Science & Information Technology program at the University of Bucharest.

## Course Structure

The course roughly follows the [curriculum](https://cs.unibuc.ro/~pirofti/ps.html) defined by [Prof. Paul Irofti](https://cs.unibuc.ro/~pirofti/). 

### Topics include:

- Introduction and basic concepts
- Time domain and systems
- Frequency domain and Fourier theory
- Low-pass and band-pass signals
- Discrete Fourier Transform (DFT)
- FIR filters
- Discrete Cosine Transform (DCT) and JPEG coding
- Sparse representations and dictionary learning
- More on dictionary learning, applications
- Wavelet Transform

## Tools & Technologies

- Language: Python 
- Libraries: NumPy, Matplotlib (for numerical work and visualization) 
- Environments: Jupyter Notebooks (also compatible with Spyder, PyCharm, VS Code)
